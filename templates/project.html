Project 1: User-Friendly Interface for Complex Application

This project involved creating a user-friendly interface for a complex application. The goal was to simplify the user experience and make the application more accessible to a wider audience.

The design process included user research, wireframing, prototyping, and user testing. The final product received positive feedback for its intuitive design and ease of use.

Below is an example of the code used in this project: 

      
        // Example JavaScript code
        function toggleMenu() {
            const nav = document.querySelector('header nav');
            const hamburger = document.querySelector('.hamburger');
            nav.classList.toggle('active');
            hamburger.classList.toggle('active');
        }

        function onScroll() {
            const elements = document.querySelectorAll('.fade-in');
            const scrollTop = window.pageYOffset || document.documentElement.scrollTop;

            elements.forEach(element => {
                const offsetTop = element.offsetTop;
                const windowHeight = window.innerHeight;

                if (scrollTop + windowHeight > offsetTop + 100) {
                    element.classList.add('visible');
                }
            });
        }

        window.addEventListener('scroll', onScroll);
        window.addEventListener('load', onScroll);
